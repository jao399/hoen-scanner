# Task 4 Design Package

This folder contains import-ready design assets and Jira ticket drafts for Task 4.

Files:
- `01-ad-types.svg`: desktop page for choosing ad types
- `02-analytics-dashboard.svg`: desktop page for ad analytics
- `03-user-survey.svg`: desktop page for user feedback questions
- `jira-ticket-drafts.md`: three Jira ticket drafts, one per design

Design direction:
- Primary dark sky: `rgb(5, 32, 60)`
- Accent sky blue: `rgb(0, 98, 227)`
- Accent eco green: `rgb(15, 161, 169)`
- Supporting surface blue: `rgb(217, 248, 255)`

These colors align with Skyscanner Backpack design tokens surfaced in Backpack docs:
- `bpk-core-primary-day`: `rgb(5, 32, 60)`
- `bpk-core-accent-day`: `rgb(0, 98, 227)`
- `bpk-marcomms-eco-green`: `rgb(15, 161, 169)`
- `bpk-marcomms-blue-muted`: `rgb(217, 248, 255)`

Suggested next click-path in Figma:
1. Create a new design file.
2. Add three pages: `Ad Types`, `Analytics`, and `Survey`.
3. Import each SVG onto its matching page.
4. Convert repeated cards into Figma components.
5. Set sharing to `Anyone with the link can view`.
6. Paste the final Figma links into `jira-ticket-drafts.md`.

Direct Figma and Jira creation were not possible from this session because no authenticated Figma MCP or Jira integration is available here.
